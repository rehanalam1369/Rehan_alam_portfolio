Vehicle Crash Detection System

PyTorch FastAPI ResNet50 Computer Vision

Computer vision model that classifies vehicle crash types with 92% accuracy.

Project Overview

This system processes input images through a custom-trained ResNet50 model that classifies vehicle crashes into front-impact or rear-impact categories with 92% accuracy.

Key Features

  • Computer vision model using PyTorch and ResNet50
  • 92% accuracy in identifying front vs. rear collisions
  • Deployed as REST API using FastAPI
  • Real-time prediction capabilities

Demo of the crash detection system in action

Technical Implementation

Model Architecture

Utilized ResNet50 architecture with custom modifications for binary classification of crash types, trained using PyTorch.

API Deployment

Implemented as a REST API using FastAPI framework, enabling real-time predictions with low latency.

Performance

Achieved 92% accuracy in testing, demonstrating reliable classification of front vs. rear collisions from images.
